### itri_lightnet class_type

0: Unknown, 
1: Red, 
2: Yellow, 
3: Green, 
4: Straight Green,
5: Straight and Right Green
6: SLR Green,
7: Red and Right,
8: Red and Left,
9: Flashing Red,
10: Flashing Yellow
